# Large-homework-code

包含《数字图像处理》课程作业所涉及到的所有代码及一些没有放入正文中的材料。

其中 `train_fashion_models.py` 为主要实验的代码。

实验结果文件说明：
- `main`：四个模型的主实验结果
- `head`：三个分类头的实验结果
- `noise`：未加入抗噪机制的实验结果
- `knoise`：加入抗噪机制的实验结果

补充实验的代码参见：[Google Colab Notebook](https://colab.research.google.com/drive/1qYSnc_-gfnTOZQWMOZNAYpwijIX7BY9Q?usp=sharing)
